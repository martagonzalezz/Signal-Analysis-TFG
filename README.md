# Signal-Analysis-TFG
This repository contains all the Python code developed for the TFG of Marta González Romero.

Functionalities:
- Data loading: functions to import and organize CSV Delsys files
- Signal processing: filtering, rectification and extraction of the sEMG linear envelope
- Signal synchronization: synchronizarion and trimming of all the signals to the same length
- ROM calculation: calculation of the elbow ROM from the IMU sensor
- Feature extraction: RMS, CI, and MNF
- Statistical analysis: intra-session and inter-session analysis performed using Linear Mixed Models (LMMs)
- Signal visualization
